# Retrospective Discipline

커밋 후 post-commit hook이 회고 체크를 알려줌. 알림이 오면 아래 기준으로 판단:

**트리거 조건** (하나라도 해당 시 `/retrospective` 즉시 실행):
- 처음 접근이 틀려서 방향을 바꿨을 때
- 예상보다 복잡한 원인 / 문서와 다른 실제 동작 발견
- 여러 실패 후 비자명한 해결책을 찾았을 때
- 프레임워크/도구의 예상치 못한 동작 발견

**트리거 안 함**: 순조롭게 완료, 단순 Q&A, 기존 지식 그대로 적용

**실행**: 제안 없이 바로 — "이번 작업을 회고해 스킬로 자산화합니다." 후 /retrospective 진행.
